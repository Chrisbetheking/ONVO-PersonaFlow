# 演示脚本

## 90 秒路径

1. **今日机会（0–15 秒）**：打开“陈女士 · L80 家庭空间咨询”，说明系统按客户意向和时效排序，而不是展示通用 KPI。
2. **内容作战台（15–40 秒）**：生成私聊、朋友圈和小红书版本；点击正文中的事实陈述，右侧定位官方来源和核验日期；点击风险表达并应用替换建议。
3. **门店审核（40–60 秒）**：保存、提交审核；经理查看完整正文、CTA、claims、风险和 evidence，填写意见并批准。
4. **跟进与试驾（60–78 秒）**：补录客户回复，展示新增记忆；打开预约弹窗，编辑时间、物品和备注，确认后时间线显示实际负责顾问。
5. **批量任务（78–90 秒）**：展示每位顾问、渠道、状态、失败原因和重试次数；执行失败项重试和抽样送审。

## 3 分钟完整路径

### 1. 机会与上下文
- 打开“今日机会”。
- 说明机会来源、为什么现在处理、客户阶段、最近消息和推荐动作。
- 点击“开始处理”。

### 2. 多平台内容与可信性
- 点击生成内容。
- 切换私聊、朋友圈、小红书。
- 编辑正文并保存草稿。
- 点击 claim，展示 evidence 来源、字段和核验日期。
- 点击 risk，展示原因和替换建议并应用。

### 3. 门店审核
- 提交审核。
- 展示完整正文和 CTA，而非摘要。
- 点击逐句高亮，查看事实与风险上下文。
- 修改正文或 CTA，填写审核意见并批准或退回。

### 4. 客户跟进与预约
- 切换到“跟进与记忆”。
- 补录客户回复，确认时间线和记忆同时更新。
- 选择许先生，打开预约弹窗，确认负责顾问为林悦。
- 编辑预约时间、携带物品和备注后提交。

### 5. 批量任务与视频边界
- 打开活动与批量任务，展示任务明细。
- 重试单个失败项，再重置演示并重试全部失败项。
- 抽样送审一条结果。
- 在内容作战台提交视频任务；未配置视频后端时只显示“保存脚本与分镜，未生成成片”。

## 断网备用路径

断开后端后刷新页面：界面会标记“离线演示数据”。重复生成、审核、补录回复、预约和批量任务流程。所有状态只保存在当前浏览器工作区，不会伪装成线上生产数据。
