# 演示脚本

## 演示前准备

- 确认页面顶部显示“演示数据 / 本地工作区 / 未连接生产系统”。
- 使用“系统治理”中的场景切换器重置当前 workspace。
- 网络正常时使用后端 Demo Adapter；网络失败时使用本地 fallback。
- 不在演示中展示任何真实密钥、真实客户或未授权平台数据。

## 90 秒版本

### 0–15 秒：定位

点击路径：`右上角角色菜单 → 总部运营空间 → 知识中心`

讲解：

> 蔚见不是文案工具。它把总部知识变化、顾问客户经营和门店人工判断连成一个闭环。

### 15–35 秒：知识变化

点击：`模拟飞书知识变更 → 调整 L80 活动结束日期`

预期：出现新版本、diff、受影响内容/客户/活动、重新核验任务和模拟通知。

讲解：

> 这里是结构化 Demo Adapter。变化会生成新版本和影响任务，不会静默改写历史内容。

### 35–60 秒：可信内容

点击路径：`切换顾问空间 → 内容作战台 → 编辑正文`

预期：事实和合规变为“需要重新核验”，提交按钮禁用；点击重新核验后恢复可提交。

讲解：

> 顾问或经理一旦改内容，旧的证据立即失效。必须由后端重新抽取事实、关联知识和执行合规后才能继续。

### 60–78 秒：客户承诺

点击路径：`跟进与承诺 → 选择承诺 → 确认/提醒/完成`

讲解：

> 沟通中的“稍后发资料”不再消失在聊天里，而是形成有截止时间和责任人的承诺。

### 78–90 秒：人工质量判断

点击路径：`切换门店经理空间 → 质量与辅导`

讲解：

> 系统只给出待复核信号和原始证据。员工可以补充上下文，最终由经理决定无需处理、提醒或辅导，不自动处罚。

## 3 分钟版本

1. **产品定位（20 秒）**：展示三个角色空间和 Demo 标签。
2. **总部知识（35 秒）**：模拟飞书变更，查看版本、diff、影响、通知和审计。
3. **热点转任务（25 秒）**：热点详情中查看证据、影响范围，创建知识草稿或内容任务。
4. **顾问内容（40 秒）**：从今日机会进入内容作战台，生成多平台内容，点击事实与风险，编辑后重新核验并提交。
5. **经理审核（25 秒）**：完整正文、CTA、claims、evidence 和 risk 并列；演示经理修改后不能直接批准。
6. **客户经营（25 秒）**：客户 360 查看判断依据和下一最佳行动；接受行动。
7. **承诺履约（20 秒）**：模拟即将到期，发送模拟飞书提醒，标记完成并进入时间线。
8. **质量辅导（20 秒）**：员工说明、经理创建辅导计划，强调人工确认。
9. **边界（10 秒）**：效果验证页显示“尚未形成真实业务结论”。

## 8 分钟完整版本

### 第一段：知识更新闭环（2 分钟）

- 总部运营空间 / 知识中心；
- 选择一条知识，查看来源、适用范围和版本；
- 模拟飞书知识变化；
- 查看旧版本过期、新版本、diff；
- 查看受影响内容、客户、顾问任务和活动；
- 对一个影响对象执行重新核验，对另一个填写忽略原因；
- 打开系统治理查看模拟通知、审批和审计。

### 第二段：客户经营闭环（3 分钟）

- 顾问空间 / 今日机会；
- 选择陈女士空间顾虑；
- 内容作战台生成私聊、朋友圈、小红书；
- 点击事实陈述定位知识；
- 点击风险并应用替换建议；
- 手工编辑正文，展示核验失效；
- 后端重新核验并提交；
- 门店经理查看完整内容并批准；
- 客户沟通中补录回复、预约家庭试驾；
- 客户档案展示状态依据和下一最佳行动；
- 承诺台账确认“准备儿童座椅”，模拟提醒并完成。

### 第三段：组织学习闭环（2 分钟）

- 门店经理空间 / 质量与辅导；
- 查看过期价格信号的原始沟通、触发规则和知识；
- 切换员工视角补充“客户引用了旧截图”的上下文；
- 经理确认创建动态事实核验辅导；
- 查看 7 天观察状态；
- 打开优秀案例，由经理确认并发布匿名案例；
- 说明案例可用于培训、FAQ 和推荐回复，但不会自动扩散未审核内容。

### 第四段：企业边界（1 分钟）

- 系统治理显示 Feishu / CRM / Messaging / Trends 均为 Demo Adapter；
- 查看审计日志和 workspace；
- 效果验证展示样本、周期和“尚未形成真实业务结论”；
- 说明生产化需要真实企业授权、数据库、RBAC 和隐私治理。

## 断网备用路径

1. 断开后端或拦截 `/api/**`；
2. 顶部出现“本地工作区 · 未连接生产系统”；
3. 从今日机会生成规则内容；
4. 编辑、事实/风险定位、保存、本地重新核验、提交和审核；
5. 补录回复、预约、记忆、承诺和批量任务；
6. 企业页面使用本地 Demo 数据并保持明确标记。

## 五个一键场景

- 飞书知识变更；
- 客户空间顾虑；
- 顾问承诺超时；
- 质量与辅导；
- 优秀案例。

切换场景只重置当前 workspace，不影响其他浏览器会话。

## v0.4.1 必演真实性节点

1. 在内容作战台修改标题或 CTA，指出顶部阻断提示和右侧证据同步变为“待重新核验”。
2. 展示“提交审核”处于禁用状态。
3. 点击重新核验，证据恢复“已核验”，再提交。
4. 在经理审核中修改正文，展示批准按钮再次被阻断，重新核验后才能批准。
5. 切换到总部运营空间的“政策与权益”，说明知识版本和有效期如何影响内容。
